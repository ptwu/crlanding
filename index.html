  • arrow_drop_downWhy a website this year?

    Mobile apps are not as accessible as an online, mobile-compatible website. This year, we are only using classreveal.com (the iOS and Android apps will not work anymore) in order to reach all students.

By using Class Reveal, you acknowledge that anyone with a wwprsd.org Google account can see the classes in which you are enrolled. We do not support non WW-P schools because of security issues. This year, we are required to verify that ClassReveal users are actual students, which is why you have to sign in using Google. We cannot verify students from other school districts, so we're limited to WW-P. Sorry for the inconvenience.
